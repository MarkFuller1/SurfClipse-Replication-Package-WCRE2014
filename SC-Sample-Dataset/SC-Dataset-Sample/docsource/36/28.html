SQLite with SQuirreL on OSX

bastian engelmann <bebass <at> web.de>
2013-09-13 19:45:36 GMT

Hello there,
 
i am trying to get SQLite run with SQuirrel on OSX 10.7.
 
The program SQuirrel itself works well. sqlite3 is part of OSX 10.7 and also works. I am trying now to add a new JDBC driver for SQLite support. Here is what i've done so far:
1.) I downloaded SQLite JDBC connector from xerial:  https://bitbucket.org/xerial/sqlite-jdbc/downloads
2.) i copied the jar into /Applications/SQuirreLSQL.app/Contents/Ressources/Java/lib
3.) i added a new driver in SQuirreL:
Name: SQLite Driver
ExampleURL: jdbc:sqlite:SQLiteDatabaseFileLocation
WebsiteURL: https://bitbucket.org/xerial.sqlite-jdbc
Class name: org.sqlite.JDBC
I chose sqlite-jdbc-3.7.2.jar from the file list
-> "OK"
4.) I receive the following notification:
Could not find class org.sqlite.JDBC in neither the Java class path nor the Extra class path of the SQLite Driver driver definition:
java.lang.ClassNotFoundException: org.sqlite.JDBC
 
As far as i can see from the source "org.sqlite.JDBC" is correct: https://bitbucket.org/xerial/sqlite-jdbc/src/417045e6cb1497b6bd7e1a3154cbcce8bd9e2854/src/main/java/org/sqlite?at=default
 
Do you have any ideas for the correct Class name?
 
Thank you in advance.
