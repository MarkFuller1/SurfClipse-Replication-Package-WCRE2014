open SQLite database in SQuirrel SQL

3 Replies

I used SQuirrelSQL at my workshop very frequently. It is a super useful open source program that can interact with your database (DB2, MySQL, SQLite..etc). To make SQLite to work with any type of database, all you need is the JDBC driver (since SQuirrelSQL is written in java) and setup the drivers correctly with the correct connection string.

In this tutorial, I will show you how to setup SQLite Driver in SQuirrelSQL. SQLite Driver is by default not in SQuirrelSQL.

Step 1: Downloading SQLite JDBC Driver

Your first step is to install SQLite JDBC connector. Go to this website to download the driverÂ (see screen shot about which file to download)

.

Step 2: Copy and Paste SQLite JDBC jar file to SQuirrelSQL bin folder.

If you install SQuirrelSQL from scratch (without changing the program folder), the SQLite program folder will most likely appear in your C:drive > Program Files (x86) > SQuirrel-sql-[version]. Open the bin folder, and you will see a tonne of jar file. Copy and paste the SQLite JDBC jar file that you have just downloaded in this location.

Step 3: Adding SQLite JDBC Driver to SQuirrleSQL

Fire up the SQuirrelSQL program, and click on the driver menu (on your left side bar) and click the add button to add a new driver in SQuirrelSQL. Enter your driver name: “SQLite Driver”, the example URL: “jdbc:sqlite:SQLiteDatabaseFileLocation” and website URL: “https://bitbucket.org/xerial/sqlite-jdbc/overview”. After that, find the SQLite JDBC Jar file that you have just added to the bin folder. (see screenshot)

Step 4: Adding your SQLite database to SQuirrelSQL

To add your SQLite database to SQuirrelSQL, go to the aliases menu (on your left side bar) and click on the add button to add new alias. Enter a name for the alias and select the SQLite driver you have just added in Step 3. For the URL field, you first need to find out the file path of your SQLite database. In this example, my SQLite database resides in my desktop folder “C:\Users\euwernNew\Desktop\test” (Note: test is my SQLite database, I did not specify the file extension for my SQLite database in this example)

Replace the “SQLiteDatabaseFileLocation” string with the actual File location of your SQLite database. (see screenshot)

Click the “Test” button and click “connect” you do not need any username or password. A dialog box will show up indicating “connection successful”.

And that’s it, you have successfully open a SQLite database in SQuirrelSQL.

One more thing, I noticed that the default installation of SQuirrelSQL comes with very tiny font size. It is not very obvious on how to increase the font size of your SQL panel, To do so, Go to File > New Session Properties. Click on the “SQL tab” and scroll down to “SQL Entry Area” and set your font size. Have fun
