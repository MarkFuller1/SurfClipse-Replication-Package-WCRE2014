JavaSpaces throws java.net.ConnectException: Connection refused exception

Hi all,

I'm trying to use the JavaSpace component as described at http://camel.apache.org/javaspace.html. I've got a GigaSpaces implementation running which also contains a "mySpace" space. I verified that the space exists and objects can be written to and read from it (by using an example project provided by GigaSpaces).

In my TestRouteBuilder, which extends RouteBuilder, I have the following code snippet to connect to the JavaSpace:

...
.to("javaspace:jini://localhost:4164?spaceName=mySpace");
...

When I start up my application, I get the following exception:

Re: JavaSpaces throws java.net.ConnectException: Connection refused exception

Hi,

Based on what I see in the component code (not an expert in Javaspaces, I'm afraid :)), it looks like the producer & consumer do not require a port, only the host.

So the URI needs to be

...
.to("javaspace:jini://localhost?spaceName=mySpace");
...

Hope this helps...
