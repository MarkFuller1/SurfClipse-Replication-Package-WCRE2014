That's a bug in the JVM - unless you are using any native code, a Java
program cannot cause such errors.

Check that you have got the latest patches for Java (and OS patches
may be relevant too).

If everything is uptodate, and you are not using native code (apart
from that which is provided by the JVM), then it might be worth
reporting the bug to the JVM provider.
