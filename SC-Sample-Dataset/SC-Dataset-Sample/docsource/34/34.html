Connection refused means you connected to a server (the proxy server) but there was no server listening on the port you used. Check you have the right host and port number.

Unless it's a type your URL is malformed. You have only one forward slash after the protocol and colon. You've written:  http:/www.yahoo.com/    versus  http://www.yahoo.com/
