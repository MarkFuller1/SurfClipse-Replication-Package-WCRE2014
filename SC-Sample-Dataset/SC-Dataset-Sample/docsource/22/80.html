java.text.ParseException: Unparseable date: "10.11.07&q

by umajava » Thu Nov 01, 2007 3:04 pm

Problem
java.text.ParseException: Unparseable date: "10.11.07"

Solution
This exception occurs while the application is trying to parse a given date.

Have a look at the below code that throws this exception

Date date = new Date();
date = DateFormat.getDateInstance(DateFormat.SHORT).parse("10.11.07");
System.out.println(date);

In the above lines of code you haven't specified the pattern the DateFormat can use. A simple way to deal with
this could be by specifying the date patten to use.

The below code works

Date date = new Date();
DateFormat sdf = new SimpleDateFormat("MM.dd.yy");
date = sdf.parse("10.11.07");
System.out.println(date);

Some times there might be an issue with your locale, in that case you can specify DateFormat as below.

DateFormat sdf = new SimpleDateFormat("MM.dd.yy", Locale.US);
