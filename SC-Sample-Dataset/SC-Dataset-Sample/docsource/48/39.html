    Hi Martin-

    A lot of unzip programs on certain platforms can be a bit deficient when dealing with files with long file names (example: class files with long names that are Eclipse plugins.)  The built-in Windows unzip program is particularly difficult with our update site archive.  So I'd recommend keeping the zip file intact and let Eclipse unzip the archive for you.

    So from the download site, save the zip file to the disk, let's assume C:\Temp\TEEPlugin-UpdateSite.zip for a moment.

    Then go to Help -> Install New Software, and at the Work with prompt, click Add.

    Enter a name (eg, "TEE 2010") and then click "Archive" and browse to the zip file you downloaded (C:\Temp\TEEPlugin-UpdateSite.zip).

    Let me know if you're still having problems.

    -ed
