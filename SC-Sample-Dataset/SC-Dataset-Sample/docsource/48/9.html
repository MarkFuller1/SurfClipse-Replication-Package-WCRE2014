eclipse can't startup when update to helios SR2

Use "help" -> "Check for Updates", selected all eclipse plugin and update.

When restart IDE, an error accrued and eclipse fail to start:

[code:java]
org.osgi.framework.BundleException: Exception in org.eclipse.osgi.framework.internal.core.SystemBundleActivator.start() of bundle org.eclipse.osgi.
    at org.eclipse.osgi.framework.internal.core.BundleContextImpl.startActivator(BundleContextImpl.java:806)

Caused by: java.lang.IllegalStateException: The System Bundle could not be resolved: Another singleton version selected: org.eclipse.osgi_3.6.2.R36x_v20110210
    at org.eclipse.osgi.internal.baseadaptor.BaseStorage.checkSystemState(BaseStorage.java:752)

Root exception:
java.lang.IllegalStateException: The System Bundle could not be resolved: Another singleton version selected: org.eclipse.osgi_3.6.2.R36x_v20110210
    at org.eclipse.osgi.internal.baseadaptor.BaseStorage.checkSystemState(BaseStorage.java:752)[/code]

Solver:

Edit file "ECLIPSE_HOME\configuration\config.ini"
change
[code:java]
osgi.bundles=reference\:file\:org.eclipse.osgi_3.6.2.R36x_v20110210.jar@4,reference\:file\:org.eclipse.equinox.simpleconfigurator_1.0.200.v20100503.jar@1\:start[/code]
to
[code:java]
osgi.bundles=reference\:file\:org.eclipse.equinox.simpleconfigurator_1.0.200.v20100503.jar@1\:start
[/code]
