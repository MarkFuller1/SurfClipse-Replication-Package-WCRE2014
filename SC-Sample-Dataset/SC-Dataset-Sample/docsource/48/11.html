    Re: Command line build

    Yes, use this command line:
    {syntaxhighlighter brush: cpp;fontsize: 100; first-line: 1; }mosyncc.exe -application com.mobilesorcery.sdk.builder.headless -data workspacelocation -project projectname [-f finalizerscript]{/syntaxhighlighter}

    Replace workspacelocation with the location of your workspace, and projectname with - believe it or not - the name of the project to build. This will actually launch a finalizer build with the currently set device filters. If you wish to produce binaries for a different set of devices you will need to create a finalizer script and use the -f switch.

    A finalizer script is just a text file containing lines like the ones produced when you click the "Propagate" button in the "Finalize" view in the IDE.
