socket API

a mechanism that provides a low level of abstraction for IPC

  

socket

a programming construct. A process wishing to communicate with another process must create an instance of such a construct. The communication between the parties may be connection-oriented or connectionless.

  

UDP

User Datagram Protocol

  

TCP

Transmission Control Protocol

  

UDP and TCP

the two key protocols at the transport layer of the Internet architecture

  

User Datagram Protocol (UDP)

allows a packet to be transported (that is, to be sent or received at the transport layer) using connectionless communication.

  

datagram

the data packet transported using connectionless communication

  

Transmission Control Protocol (TCP)

is connection-oriented and transports a stream of data over a logical connection established between the sender and the receiver.

  

datagram sockets

sockets that use UDP for transport

  

stream sockets

sockets that use TCP

  

Java classes for the datagram socket API

the DatagramSocket class for the sockets, and the DatagramPacket class for the datagrams exchanged

  

bound

Each socket is said to be _____________ to a UDP port of the machine that is local to the process.

nonblocking

In basic socket APIs, whether connection-oriented or connectionless, the send operations are _________________ while the recieve operations are blocking.

  

setSoTimeout method

a method to set a timeout for some interval to avoid indefinite blocking

  

void connect(InetAddress address, int port)

Java method that creates a logical connection between this socket and a socket at the remote address and port

  

void disconnect()

Java method that terminates the current connection, if any, from this socket

  

connection sockets and data sockets

the two types of sockets with the stream-mode socket API

  

connection sockets

provided via the ServerSocket class, is for accepting connections

  

data sockets

provided via the Socket class, is for data exchange

  

server

the process that establishes a connection socket and then listens for connection requests from other processes

  

client

the process that wishes to communicate with the server

  

ServerSocket and Socket

the two main classes in the stream socket API

  

ServerSocket(int port)

key method/constructor of the ServerSocket(connection socket) class that creates a server socket on a specified port

  

Socket accept() throws IOException

key method/constructor of the ServerSocket(connection socket) class that listens for a connection to be made to this socket and accepts it. The method blocks until a connection is made.

  

void close() throws IOException

key method/constructor of the ServerSocket(connection socket) class that closes this socket

  

void setSoTimeOut(int timeout) throws SocketException

key method/constructor of the ServerSocket(connection socket) class that sets a timeout period (in milliseconds) so that a call to accept() for this socket will block for only this amount of time. If the timeout expires, a java.io.InterruptedIOException is raised

  

Socket(InetAddress address, int port)

key method/constructor of the Socket(data socket) class that creates a stream socket and connects it to the specified port number at the specified IP address

  

void close() throws IOException

key method/constructor of the Socket(data socket) class that closes this socket

  

InputStream getInputStream() throws IOException

key method/constructor of the Socket(data socket) class that returns an input stream so that data may be read from this socket

  

OutputStream getOutputStream() throws IOException

key method/constructor of the Socket(data socket) class that returns and output stream so that data may be written to this socket

  

void setSoTimeout (int timeout) throws SocketException

key method/constructor of the Socket(data socket) class that sets a timeout period for blocking so that a read() call on the InputStream associated with this socket will block for only this amount of time. If the timeout expires, a java.io.InterruptedIOException is raised
