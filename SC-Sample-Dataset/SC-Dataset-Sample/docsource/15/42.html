Oracle eJRE for Medallion Touch‑Screen Computer (Module)

Techsol offers Oracle’s eJRE (embedded Java Runtime Environment) for use on the Medallion Touch‑Screen Computer (Module). Oracle’s Java SE Embedded eJRE is an embedded version of Java SE that has been adapted to fit into a smaller footprint than the desktop version.

For more information, please see Techsol’s associated press release.

IBM J9 JRE for SA2410 Medallion Classic CPU Module

Techsol offers IBM's J9 JRE (Java Runtime Environment) for use on the SA2410 Medallion Classic CPU Module. J9 is an implementation of J2ME/CDC.

Techsol’s port of J9 includes the following classes and API packages:
