Gen of Stubs for Jasper Server with axis-wsdl2java

Posted on June 24, 2008 at 4:26pm
0
Hi All,

I am trying to Generate Jasper Server Stubs using axis-wsdl2java ant task as:

<target name="genstub">
<axis-wsdl2java output="${stub.home}" url="${wsdl.url}"/>
</target>

where wsdl.url is ::
wsdl.url=http://IVP-PC-148.ivp.co.in:8080/jasperserver/services/repository?wsdl

But i am getting the following Error:
genstub:
[axis-wsdl2java] WSDL2Java http://IVP-PC-148.ivp.co.in:8080/jasperserver/services/repository?wsdl
[axis-wsdl2java] java.net.ProtocolException: Server redirected too many times (20)

Please let me know if I am using the wrong URL or Guide me how to get the API for Jasper Server to access it..

NOTE: The ANT task is working as i checked it for other site which is:
http://compName:8000/wsdl/v7/axis/all

Please Help

Thanks in Advance,

jagsisingh wrote:

[axis-wsdl2java] WSDL2Java http://IVP-PC-148.ivp.co.in:8080/jasperserver/services/repository?wsdl
[axis-wsdl2java] java.net.ProtocolException: Server redirected too many times (20)


This might be caused by the fact that authentication is required to access the URL.

You could try one of the following:
  • Put authentication tokens in the URL, i.e.
    http://user:password@IVP-PC-148.ivp.co.in:8080/jasperserver/services/rep...


  • Download the WSDL using a browser, and use the local copy for wsdl2java.
    • [/ul]
