Increasing Java Heap Space

Ever see this (or something similar)?
ERROR
CenterView Server error: Java heap space

Full Error: java.util.concurrent.ExecutionException: java.rmi.ServerError: Error occurred in server thread; nested exception is: java.lang.OutOfMemoryError: Java heap space

No preview is available.
 
You've run out of Memory.

One way to fix the situation is to increase your Tomcat settings (when operating on a standard install environment).

  1. locate your setenv.sh file under <INSTALL DIRECTORY>/Tomcat/bin/ 
  2. open it in a text editor
  3. locate the line below:
    • JAVA_OPTS="-Xms512m -Xmx512m -XX:MaxPermSize=128m -XX:+DisableExplicitGC -XX:+UseConcMarkSweepGC -Dsun.rmi.dgc.client.gcInterval=360000 -Dsun.rmi.dgc.server.gcInterval=360000"
  4. the -Xms setting is your initlal memory setting (in MB), while the -Xmx is the maximum.  increase the maximum to your desired maximum (being careful not to allocate so much that the rest of your machine suffers).  1024 is a good start:
    • JAVA_OPTS="-Xms512m -Xmx1024m -XX:MaxPermSize=128m -XX:+DisableExplicitGC -XX:+UseConcMarkSweepGC -Dsun.rmi.dgc.client.gcInterval=360000 -Dsun.rmi.dgc.server.gcInterval=360000"
  5. save the file
  6. restart your server

NOTE: it is also possible to do the same thing by updating the JAVA_OPTS env variable.
