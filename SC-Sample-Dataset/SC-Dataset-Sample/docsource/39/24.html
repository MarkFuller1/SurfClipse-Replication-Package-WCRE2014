Weird because the ERjavaMail/resources/Properties file defines this:

# * Default encoding used for ERJavaMail
er.javamail.defaultEncoding = iso-8859-1
You can try overriding that in your app Properties with UTF8 and see how that works for your environment. 
er.javamail.defaultEncoding = UTF-8
