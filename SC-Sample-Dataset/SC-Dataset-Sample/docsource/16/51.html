Java error: java.net.MalformedURLException: unknown protocol: socket

Posted by Charles on May 5, 2011 Leave a comment (3) Go to comments

I got 

Java error: java.net.MalformedURLException: unknown protocol: socket

error when I ran the scottrade Streaming quotes jnlp. Took me some googling to find the answer

Run “jcontrol” from command line
General > Network Setting > Direct Connection
