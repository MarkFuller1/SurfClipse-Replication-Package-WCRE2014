I believe I've performed the JSSE installation process correctly, but I'm still unable to run even the simplest of programs, e.g. one of the API's sample applications:

code:

import java.net.*;
import java.io.*;
public class URLReader {
    public static void main(String[] args) throws Exception {
	System.setProperty("java.protocol.handler.pkgs", "com.sun.net.ssl.internal.www.protocol");
	URL verisign = new URL("https://www.verisign.com/");
	BufferedReader in = new BufferedReader(
				new InputStreamReader(
				verisign.openStream()));
		String inputLine;
		while ((inputLine = in.readLine()) != null)
			System.out.println(inputLine);
		in.close();
    }
}

The code above compiles, but I get a MalformedURLException - "unknown protocol: https" - at runtime. Any ideas as to what I'm doing wrong?

By the way, I'm registering the SunJSSE provider statically, i.e. in my java.security file, so the lack of a call to Security.addProvider(...) shouldn't be the problem.
