Filtering categories in Organiser dialog

by j0lxj0lx » Thu Mar 29, 2012 4:13 pm

CC can tag the Project name with a prefix character like "#" and make it an Outlook category

It can also tag contexts with @ and other things (like next action with "!").

However, in the Organiser dialog box, it shows ALL these categories, inluding anything with a special symbol on the front.

Unfortuantely, if I have an item tagged with #Project, @Category and !Next, it shows up THREE times.

Anbd since I have hundreds of projects (in my line of work), it takes forever to wade through the project categorised by "#" to get to the @Context tags right at the bottom.

Does anybody know how to configure CC to show tasks sorted and displayed by ONLY categories with an '@' symbol, and ignore the rest of the symbols?

Re: Filtering categories in Organiser dialog

by ClearContext Support » Fri Mar 30, 2012 10:16 am

While ClearContext considers all categories that start with a non-alphanumeric to be a context category, those that start with the special Project prefix should be ignored.

It looks like you found a bug in the category view of the Organizer when the suffix for Project categories is not set. We will get this fixed in the next update.
