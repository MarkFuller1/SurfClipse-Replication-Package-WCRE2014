InputMismatch Exception Handling

Posted 05 February 2009 - 05:41 AM

Hey all, been trying to run this code which is a exception handling task and the code works but my catch isnt working properly. Task is to take in any random number of positive integers given by user, then when user enters a non-integer it should display the sum and quit. Sorry if this is basic, im visiting Amsterdam but just cant help myself trying to solve this one I have gone through reference sheets, try tutorials and the like but just cant see it, it may be how I am using my brackets? Am using Eclipse for RCP/Plug-in Developers to program and run my java apps. Thanks!

import java.util.*;
//import of scanner utility
public class Sum {

	public static void main(String[] args) {
		
		//Setting up scanner, boolean and initializing variables
		Scanner scanner = new Scanner(System.in);
		boolean valid = true;
		int sum = 0;
		int a = 0;
		
	while (valid){
		//While loop, always runs once since valid=true
		
		//Collection of input integers
		System.out.print("Please enter a positive integer, enter a non-integer to sum: ");
		a = scanner.nextInt();
		
					//'a' is now entered integer
		try{
			//Case for 'a' being positive integer and valid
			if (a >= 0 && valid){
				sum = sum + a;
			}
			//Case for 'a' being negative
			else if (valid){
				System.out.println("Please enter only positive integers!");
			}
			
			//While loop runs again here
			valid = true;
		}
		//What to do when input is not integer
		catch (InputMismatchException e){
			
			//This code should display sum and set valid=false, breaking the while loop
			System.out.println("The sum of your integers is: "+sum);
			valid = false;
		}
		//Doesn't need to be here, just to show use of all parts of try
		finally{
			System.out.println("Num Num Numbers!");
			
		}
		}
	
}
}


After entering a non-integer Eclipse throws the InputMismatchError in 'main' and looks like this in the console;
Please enter a positive integer, enter a non-integer to sum: 8
Num Num Numbers!
Please enter a positive integer, enter a non-integer to sum: 9
Num Num Numbers!
Please enter a positive integer, enter a non-integer to sum: z

	at java.util.Scanner.next(Unknown Source)
	at java.util.Scanner.nextInt(Unknown Source)
	at java.util.Scanner.nextInt(Unknown Source)
	at Sum.main(Sum.java:18)
