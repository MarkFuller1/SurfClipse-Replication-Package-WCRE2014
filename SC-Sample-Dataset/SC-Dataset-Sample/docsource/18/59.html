                I was able to connect to WAS6.1 with these libraries on my classpath:

                I extracted these from sibc_install*.jar (download at http://www-01.ibm.com/support/docview.wss?uid=swg24012804):

                sibc.jms.jar
                sibc.jndi.jar
                sibc.orb.jar

                And these are from WAS:

                com.ibm.mqjms.jar
                com.ibm.mq.jar
                dhbcore.jar
                com.ibm.ws.webservices.thinclient_6.1.0.jar
                com.ibm.ws.runtime_6.1.0.jar

                And add system property:
                -Dcom.ibm.CORBA.ORBInit=com.ibm.ws.sib.client.ORB

                This was working for me on Sun Java 1.6.0_11 but should on 1.5 also
                Cheers
