The exception is unable to

The exception is unable to find this field:

JMemory.physicalSize:int

This field doesn't exist in 1.3.b4 anymore. This means that you are still linking against the wrong/old jnetpcap.dll file. An older version of the dll must be either in the PATH variable, \windows\system32 or if you define a -Djava.library.path property.
