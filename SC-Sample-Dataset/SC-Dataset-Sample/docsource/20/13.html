Are you using the LibSVM package for Weka >= 3.7.2? If so, this error
should not occur because the libsvm.jar file bundled with the package
has been updated to version 3.12. The only reason I can think of for
this exception to occur is that you have an older version of libsvm.jar
in your classpath somewhere.
