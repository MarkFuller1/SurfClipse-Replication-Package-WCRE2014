  2. fredt

    2011-06-26 18:09:35 PDT
    You are trying to return a ResultSet from a stored procedure. This is not supported in version 1.8.x Please use the latest version 2.2.4 which supports this, and uses a different SQL syntax with CREATE PROCEDURE instead of CREATE ALIAS. See the guide here: http://hsqldb.org/doc/2.0/guide/sqlroutines-chapt.html which has examples.

  8. fredt

    2011-08-21 12:48:25 PDT
    A stored procedure returns values in OUT parameter. The example you mention actually calls a stored procedure with a single OUT parameter. The parameter value can then be retrieved using one of the getXXX() methods of CallableStatement. You can try with the current release or the snapshot jar on the support page and define a procedure with one OUT parameter and call it.
