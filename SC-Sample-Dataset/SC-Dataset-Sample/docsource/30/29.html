Offline Gradle Mode
The next release of Android Studio will support offline mode when building a project ("offline" mode allows a build to operate without accessing network resources.) To enable this feature (it is disabled by default) go to the Preferences page ("Settings" in Windows) and navigate to the "Compiler" > "Gradle" page (see screenshot 1.) In addition, if unresolved dependencies are detected while building a project in offline mode, Android Studio will suggest you to go back to "online" mode (see screenshot 2.)
Work is still needed to support offline mode when creating/importing/syncing projects.﻿
