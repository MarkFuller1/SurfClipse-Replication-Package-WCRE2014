In response to post #9606045.

I also had this problem but with ASIS patcher. This issue is from a bad edit for practice arrows in hunting in skyrim mod. The author of that mod has been informed and should be fixed in next update. This happens because Reproccer looks for the masters on the list, but since AV has aborted it will not find it.

I never used AV but I think it's the same issue. So you must include hunting in skyrim esp in the block-list of AV to exclude this mod and let reproccer finish correctly.

you should edit block-list of AV (look under data\SkyProc Patchers in game folder). If that is too difficult before patching enter your game launcher/mod manager and disable hunting in skyrim. Launch patcher and when finished patching, re-enable hunting in skyrim. remember to correct your load order after this process.

PD if you are using SkyRE you shound be using hunting in skyrim no perks version for compatibility.
