Nicolas Modrzyk wrote:


Hi Marc,

The changes are in CVS.
You can now specify the jmx adaptors you need and on what port you want them to be assigned. The "JmxSettings" section on the cjdbc-controller.xml file has changed. You should now use 
something like:

<JmxSettings enabled="true">
 <jmxAdaptor type="rmi" port="1099"/>
 <jmxAdaptor type="http" port="8888"/>
</JmxSettings>
You can just remove any adaptor if you don't need them, but be careful that if no rmi adaptor is specified, 
you cannot start the console since it works through jmx over rmi.

DTD, and examples have been updated as well.

Nicolas,
