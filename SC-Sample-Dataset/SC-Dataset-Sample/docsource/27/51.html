Hi Chad,

Welcome to Twitter programming. Is Java the language you're most comfortable with? twitter4j is a great library but it abstracts away a lot of the nitty gritty of working with the API itself.

I'm not sure what you're trying to accomplish with your code, but the exception seems to indicate that the code is having trouble even connecting to Twitter's servers -- I'd make sure the machine you're executing this on is able to communicate via SSL to api.twitter.com before proceeding.
