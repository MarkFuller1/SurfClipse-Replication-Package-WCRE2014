Exception handling in java

Posted on June 2, 2013 by Ajantha Standard

Exceptions
Handle runtime error
Avoid breakdown of java program

Compile-time error
- cannot create class file

Run-time error
- results in program breakdown
- need to handle runtime erros using OOPS

Exception
- is a Class
- All runtime errors generate an exception object

Syntax for handling exceptions

try
{
  // statements
}
catch (ExceptionType object)
{
  // User error handling
}

Example
Program to illustrate why we need exception handling
 

// Find cube of a number
class c13_1
{
	public static void main(String[] args) 
	{
		int n,c;
		n=Integer.parseInt(args[0]);
		c= n*n*n;
		System.out.println(c);
	}
}

Example

// Find cube of a number
class c13_1
{
	public static void main(String[] args) 
	{
		int n, c=0;
		try
		{
			n= Integer.parseInt(args[0]);
			c = n*n*n;
		}	
		catch(Exception e)
		{
			System.out.println("Exception caught ! Error in data ");
		}
		System.out.println(c);
	}
}

Output
