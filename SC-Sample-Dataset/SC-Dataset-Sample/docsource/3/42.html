The best way to see the format that is required is to use the cost matrix editor from the Explorer (ClassifyPanel-->More options…) and then save the matrix to a file. E.g. For two classes this is the format that is expected:

% Rows  Columns
2       2
% Matrix elements
0.0     1.0     
1.0     0.0 

Cost values should be positive too.
