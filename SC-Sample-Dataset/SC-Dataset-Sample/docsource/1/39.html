It's the path on disk that is important.  Your profile needs to physically end up in a different directory so that there is no directory with ! in it.
I know that people have had success by creating a new account.
You might also find success by using the portable minecraft technique.

A bat file that contains something like:
set APPDATA=<pathtosomewhere>\data        
<pathtominecraft>\minecraft.exe

You'd need write access, to the directory you set APPDATA to.
