Open the admin.config file and comment the following line
com.ibm.ejs.sm.adminServer.logFile=<filename>,<filename>
and then start the admin server it will work fine
the logFile parameter of the com.ibm.ejs.sm.adminServer package specifies the files to be used to record administrative server events. just comment it and try starting it
