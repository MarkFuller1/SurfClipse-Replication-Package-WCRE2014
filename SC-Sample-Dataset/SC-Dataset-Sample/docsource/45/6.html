leleuj commented March 18, 2013

It looks like you have a lot of troubles with your SSL local configuration. The service ticket validation between the application and the CAS server requires a direct communication which might be using HTTPS if you define your CAS server url that way. Though, if your CAS server supports both (HTTP & HTTPS), you can validate the service tickets using HTTP by defining a HTTP url for the casPrefixUrl property.
