Try to generate the stub as follows. Please use http port instead of https one. $AXIS2_HOME//bin/wsdl2java.sh -uri http://localhost:8080/axis2/services/GreetingService?wsdl <https://localhost:8443/axis2/services/GreetingService?wsdl> -p test.wstest.client -uw -o 

regards
Charitha
