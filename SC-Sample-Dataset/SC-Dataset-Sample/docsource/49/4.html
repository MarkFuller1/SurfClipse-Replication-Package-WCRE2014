Resolving the problem

Remove or correct the duplicate entry in extended dimensions


Steps:

1. Look for the duplicate extended dimension in the table xdimnyst
a) Maintain, Special Utilities, Browse Data, xdimnyst
b) tkod = 'ABC'
c) typ indicates which extended dimension structure (1-4) this item is found into

2. Delete and/or correct the duplicate entry
a) Maintain, Select the appropriate ext structure (1-4)
b) Find the duplicates items on the same structure
c) Delete or correct accordingly

3. Logout and log back into Controller

4. Load the desired menu option
