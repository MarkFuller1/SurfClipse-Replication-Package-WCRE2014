  13. viakia

    Hay everyone.

    I too had this happen to me, but i sort of found a loophole on saving your game data.

    Open your "server" properties file and change Level-name= to a different world name, than save and run. Once all the files are created, open the main files for both worlds. I replace the new: Data, Players, Region, and Level .dat file with the old ones. Make sure not to copy the old Railcraft.dat and the RedstoneEther files, because I found these to be the corrupted ones.

    once its all copied over run the server and try to log in. If your able to load the server, destroy all anchors.
    If you weren't able to load it try temporarily removing the Railcraft.dat, log and destroy anchors than reinsert .dat

    I cant promise it will work or fix it for long but as far as i'm able to tell my level is back to normal.


    Hope this helps
    viakia, Oct 13, 2012
