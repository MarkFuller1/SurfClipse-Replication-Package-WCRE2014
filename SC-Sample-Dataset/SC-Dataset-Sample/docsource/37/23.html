Finally, I got the problem. My bad..
Looks like, Ajax behavior works within an h:form. While trying to fix the original bug; I downloaded a sample application from else where and tried to append the code that caused the problem. The server, when tried to run it gave a hint by saying:

Code: Select all
HTTP Status 500 - AjaxBehavior for : "counter" must be inside a form element

--------------------------------------------------------------------------------

type Exception report

message AjaxBehavior for : "counter" must be inside a form element

description The server encountered an internal error (AjaxBehavior for : "counter" must be inside a form element) that prevented it from fulfilling this request.


I put it in an h:form with an h:commandButton and it worked!!
