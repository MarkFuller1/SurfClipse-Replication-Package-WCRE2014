marchof commented January 31, 2013

Looks like a invalid class in sqlite. As you probably don't want to measure code coverage of the sqlite library itself you can exclude these classes in the global EclEmma preferences. For this add "org.sqlite.*" to the excludes, see documentation: http://www.eclemma.org/userdoc/preferences.html
