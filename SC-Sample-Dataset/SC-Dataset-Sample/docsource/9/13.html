marchof commented January 10, 2013

Actually this is a problem with the class file: JaCoCo expects Java 6 class files to contain proper stackmap frames. Unlike the JVM JaCoCo has no fallback for invalid or missing stackmap frames (which should be present for Java 6 class files).

The class file you provided is version 6 but does not contain stackmap frames at all. Please report this to the tool provider which preprocesses your class files.
