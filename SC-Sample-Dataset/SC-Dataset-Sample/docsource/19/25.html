    on the following code

    StringTokenizer t = new StringTokenizer(Mystring, ",");
    String indexn = t.nextToken();
    String Description = t.nextToken();
    Double indexvalue = Double.parseDouble(t.nextToken());

    on these input format:
    6,"10 YEAR TREAS",4.74

    However the code worked OK on these format (without quotations):
    6,10 YEAR TREAS,4.74

    i believe that you need to check the flag returned from hasMoreToken() or hasMoreElement() before you call nextToken().

    only call nextToken() if hasMoreToken() returns true.

    hasMoreElement() works the same way as hasMoreToken(), it is there so that the tokenizer can implement the Enumeration interface.
