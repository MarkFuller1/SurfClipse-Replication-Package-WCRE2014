Hi user,

Thanks for replying. So you mean with the parameter -Dsun.java2d.noddraw=false in "run.bat" the application is started properly?

If so, please follow the steps below for applying the parameter to VP-UML launcher:
1. shutdown VP-UML
2. go to %{VP_installation_dir}\bin folder
3. create a blank file, name it "Visual Paradigm for UML 7.1.vmoptions" (.vmoptions is the file extension)
4. edit the file and enter
-Dsun.java2d.noddraw=false
5. save the file
6. go to %{VP_installation_dir}\launcher folder
7. create a blank file, name it "run_vpuml.vmoptions"
8. repeat steps 4 & 5 on the file

After these steps, you will be able to start VP-UML by launcher/Start Menu/shortcut. Please feel free to ask if there is anything I can do to help.
