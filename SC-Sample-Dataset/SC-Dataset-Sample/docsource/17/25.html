Server code running through RMI - 

package Bank_DO;

import java.net.MalformedURLException;
import java.rmi.*;
import java.rmi.Naming;
import java.rmi.RemoteException;

import Bank_DO.Stock;

public class StockServer {
    
    public static void main(String[] args) throws RemoteException, MalformedURLException
    {
        System.setSecurityManager(new RMISecurityManager());
        
        Stock s = new Stock();
        Naming.rebind("Stock", s);
    }
}



Client code to run and connect to server - 

package Bank_DO;

import java.net.MalformedURLException;
import java.rmi.Naming;
import java.rmi.NotBoundException;
import java.rmi.RemoteException;

public class StockClient
{
     public static void main (String[] args) throws MalformedURLException, RemoteException, NotBoundException
     {
        String url = "rmi:&#47;&#47;localhost&#47;Stock";
        
        IStock stock = (IStock)Naming.lookup(url);
        
        stock.setCompanyName("AJAX Inc.");
        System.out.println(stock.showShare());
        
     }   
}



Afterwards, compile all the files. Then run this command (with change as needed):
rmic -classpath D:\program\JCreatorV3_LE\MyProjects\Bank_DO -d D:\program\JCreatorV3_LE\MyProjects\Bank_DO Bank_DO.Stock

This creates the stub and skeleton .class files for the RMIregistry.

You'll see in StockServer there's a call to RMISecurityManager. From what the errors I've gotten and Google, you'll need to set a new RMISecurityManager and create two .policy files and put it in the same classpath (directory for UNIX or folder for Windows) as the server and client .class files. They can be made in notepad (or any text editor) and saved as xxx.policy.

here are the .policy files:
//S_Stock.policy
grant codeBase "file:/D:/program/JCreatorV3_LE/MyProjects/Bank_DO/" {
    permission java.security.AllPermission;
};

//C_Stock.policy
grant codeBase "file:/D:/program/JCreatorV3_LE/MyProjects/Bank_DO/" {
    permission java.security.AllPermission;
};

For my example, the code is that same because it is all the same folder.

Note: commands below are meant to be one line.

Now run the Server class - 
D:\program\j2sdk1.4.2_03\bin>java -Djava.rmi.server.codebase=file:\D:\program\J
reatorV3_LE\MyProjects\Bank_DO\ -Djava.security.policy=D:\program\JCreatorV3_LE
MyProjects\Bank_DO\Bank_DO\S_Stock.policy -cp D:\program\JCreatorV3_LE\MyProjec
s\Bank_DO\ Bank_DO.StockServer


Finally the Client class - 
D:\program\j2sdk1.4.2_03\bin>java
 -Djava.rmi.server.codebase=file:\D:\program\JCreatorV3_LE\MyProjectsBank_DO -Djava.rmi.server.hostname=127.0.0.1
-Djava.security.policy=D:\programJCreatorV3_LE\MyProjects\Bank_DO\Bank_DO\C_Stock.policy
