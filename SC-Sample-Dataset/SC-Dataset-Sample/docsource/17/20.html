    Java Code: 
    System.out.println("Exception "+e+"Raised")
    This tells you next to nothing about the problem.
    Java Code: 
    e.printStackTrace();
    this will tell you exactly what exception has occurred and, more importantly, where it occurred.

    Always print the stack trace on an exception.
