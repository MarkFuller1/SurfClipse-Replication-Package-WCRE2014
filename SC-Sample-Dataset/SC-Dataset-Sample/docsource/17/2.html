    After you successfully lookup the EJB Home, you must narrow it:

    //...
    Object obj = initialContext.lookup(jndiName);
    Home home = (Home) PortableRemoteObject.narrow(obj, Home.class);
    Session session = session.create();

    This is only necessary for remote lookups.

    Cheers and happy coding,
    Martin
