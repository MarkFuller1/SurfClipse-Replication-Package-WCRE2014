Problem Note 44434: SASÂ® Marketing Automation's maextract utility might return a "java.lang.NoSuchMethodException" error on a campaign that contains a prioritize node

The maextract command in SAS Marketing Automation's Integration Utilities might write the following error to the auxdataio extract log when you extract a campaign: 

com.sas.analytics.crm.flow.client.PrioritizeNodeLineItemDO.getMemberTableDO()
java.lang.NoSuchMethodException:
com.sas.analytics.crm.flow.client.PrioritizeNodeLineItemDO.getMemberTableDO()
	at java.lang.Class.getMethod(Class.java:1581)

This error is written only when you extract a campaign that contains a prioritize node. The error can safely be ignored.
