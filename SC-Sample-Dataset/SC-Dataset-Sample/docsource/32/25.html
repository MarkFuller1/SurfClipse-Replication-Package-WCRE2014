Chandika,

Did you check the section jaxws on Axis2's site [1]? 
Using annotations and then deploy your jars on the servicejar directory  you can publish your services
using annotations (no xml descriptor needed). Take a look to this how-to [2] in order to have an idea. Then,
on the client side Axis2 lets you consume the exposed services using generated stubs (wsdl2java) or
direclty with the ServiceClient API [3,4].
