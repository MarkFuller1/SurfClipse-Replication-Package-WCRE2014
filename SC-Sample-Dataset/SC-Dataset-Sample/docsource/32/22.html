COMPLETE FIX!!!!

Hi does anyone have a link to the Aion 3.0 client? i have the 3.7 but when i try to connect it logs in, selects server but then it disconnects me, is this because I'm trying to connect with a 3.7 client?

***UPDATE: Anyone still looking for a downgrade from 3.5.0.12(aka 3.7) to 3.0.0.8or3.0.0.4. I found a solution. With the core posted and the files i downloaded I managed to downgrade and run my server. Furthermore, I managed to set it up via Hamachi so that friends could connect too.

Needed Files:
Downgrade link - [Only registered and activated users can see links. ]
White Glow Fix for the HD graphics engine - [Only registered and activated users can see links. ]
Fix for Windows 8 users - [Only registered and activated users can see links. ]

Note: The white glow is for the downgrade, when u set your graphix settings to high on the new engine, your character sometimes gets a white glow around it(no its not what you think, its ugly).

To get it working via Hamachi. In the Bat file, replace the 127.0.0.1(localhost) with your hamachi ip.

In the C:\AionServer\AIONEmulator\game\config\ipconfig.xm l

Change the default ip to your Hamachi Ip as well.

You .bat File should look like this:
Code:
@echo off
echo Starting Aion ...

if exist L10N\1_enu\1_enu.pak (
  start bin32\aion.bin -ip:your hamachi ip -port:2106 -cc:1 -noauthgg -ls -charnamemenu -ncg -ingameshop -lang:enu -noweb -DEVMODE "con_disable_console = 0" -DEVMODE "g_freefly 1" -DEVMODE "g_chatlog 1" -DEVMODE "log_IncludeTime 1" -DEVMODE "log_Verbosity 1" -DEVMODE "log_FileVerbosity 1"
) else if exist L10N\1_deu\1_deu.pak (
  start bin32\aion.bin -ip:your hamachi ip -port:2106 -cc:2 -lang:deu -noweb -nokicks -ncg -noauthgg -ls -charnamemenu -ingameshop -DEVMODE "con_disable_console 0" -DEVMODE "g_chatlog 1" -DEVMODE "g_auto_disconnect 0"
) else if exist L10N\1_fra\1_fra.pak (
  start bin32\aion.bin -ip:your hamachi ip -port:2106 -cc:2 -lang:fra -noweb -nokicks -ncg -noauthgg -ls -charnamemenu -ingameshop -DEVMODE "con_disable_console 0" -DEVMODE "g_chatlog 1" -DEVMODE "g_auto_disconnect 0"
) else if exist L10N\1_rus\1_rus.pak (
  start bin32\aion.bin -ip:your hamachi ip -port:2106 -cc:7 -lang:rus -noweb -nokicks -ncg -noauthgg -ls -charnamemenu -ingameshop -DEVMODE "con_disable_console 0" -DEVMODE "g_chatlog 1" -DEVMODE "g_auto_disconnect 0"
) else (
 echo Aion folder not found or unsupported language.
 pause. 
)
Tell your friends to paste it in the root of their Aion folder, and make a shortcut to their desktop.

The XML should look like this:

Code:
<ipconfig default="Your Hamachi IP">
	<iprange min="10.0.0.0" max="10.255.255.255" address="10.0.0.0"/>
	<iprange min="172.16.0.0" max="172.31.255.255" address="172.16.0.0"/>
	<iprange min="192.168.1.0" max="192.168.255.255" address="192.168.1.0"/>
	<iprange min="127.0.0.1" max="127.0.255.255" address="127.0.0.1"/>
</ipconfig>
Use notepadd++ to edit it or just your normal wordpad, they do the same.

One final thing if they(your friends) are unable to connect, They can edit their config.ini file located at: C:\Program Files\NCSoft\Aion\bin32\config.ini and replace the Ip's with your Hamachi IP.

The Config.ini file should look like this:
Code:
[SET_DNS_TCP_CLIENT]
	BIND_ADDR = Your Hamachi IP

[ServerAddr]
	BIND_ADDR = Your Hamachi IP
	BIND_PORT = Same port as the bat launcher
If they Still cant connect, chek your firewall settings for inbound connections,
and make sure you open your router ports as follow:
Code:
Application	Trigger	Open	Remove
Name	Protocol	Port Range	Protocol	Port Range	 
 	 	        Start	End	 	        Start	End	 
Aion	TCP/UDP	7777	7777	TCP/UDP	7777	7777	
Aion	TCP/UDP	2106	2106	TCP/UDP	2106	2106	
Aion	TCP/UDP	8096	8096	TCP/UDP	8096	8096

My main reason for posting this update is because i could not find a complete guide to server and client side setup. I give you permission to use this info and share this info in any way u'd like.
