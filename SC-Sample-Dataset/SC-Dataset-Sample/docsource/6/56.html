You need some code like this:

char str[16]; // use an uninitialized array
FILE *fp; // file pointer
fp = fopen("<filename>", "rb"); // open file in read bytes mode
fread(str, sizeof(char), 16, fp); // read 16 elements from file, each sizeof(char) bytes wide (sizeof(char)=1)
fclose(fp); // close file

If you have each character on a new line, it's better to treat the file as text rather than binary, so this approach works better:

char str[16];
FILE *fp;
fp = fopen("<filename>","r"); // open file in read mode
for (int n = 0; n < 16; n++) { // loop through characters you want to read
    fscanf(fp, " %c ", str + n); // *remark
}

remark: " %c " is a format specifier which says to consume all whitespace until first non-whitespace character, read a character, and then consume all whitespace again. The character read is stored into the pointer given as the final argument, which is a pointer to str advanced n elements forward. str[n] would not work because this is the value at n, not a pointer to the element at n.

You should probably read this: http://www.cplusplus.com/reference/cstdio/fscanf/
